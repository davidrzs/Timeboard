# Timeboard

A kanban-style task manager that helps you organize work by time horizon. Tasks flow naturally from Backlog through Later, Next Week, This Week, and Today columns based on their due dates.

## Features

- **Time-based kanban board** - Tasks automatically move between columns based on due dates
- **Project grouping** - Switch between time horizon and project views
- **Google Calendar integration** - See your calendar events alongside tasks
- **Natural language input** - Add tasks like "Call mom tomorrow" or "Report due next Friday #work"
- **Drag and drop** - Easily reschedule tasks by moving them between columns

## Tech Stack

- **Backend**: Django 6 + Django Ninja, PostgreSQL
- **Frontend**: SvelteKit (Svelte 5), Tailwind CSS 4
- **Auth**: Google OAuth
